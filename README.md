# 🎯 AI Resume Analyzer

> **Claude-Powered Resume & Job Fit Analyzer — Instant ATS scoring, skill gap analysis, and tailored career recommendations**

AI Resume Analyzer compares any resume against a job description — extracting skills, computing ATS coverage, scoring fit across 4 dimensions, and streaming personalized recommendations powered by Anthropic Claude.

## 🧠 What is AI Resume Analyzer?

AI Resume Analyzer is a **3-stage AI pipeline** that helps candidates and recruiters understand resume-job fit in seconds:

1. **Parse** — Extracts structured data from any PDF/DOCX resume (name, skills, experience, education)
2. **Score** — Deterministically computes 4 fit scores (skills, experience, education, ATS) before any AI call
3. **Recommend** — Claude streams personalized, role-specific recommendations to improve the resume

Unlike generic resume tools, scores are computed from actual document signals — not generated by AI — so they're consistent and unbiased.

```
║                                                                      ║
║   Resume side:                       JD side:                        ║
║   resume_service.py                  job_service.py                 ║
║                                                                      ║
║   1. extract_text():                 1. Receive raw JD text          ║
║      .pdf  → PyMuPDF                                                 ║
║      .docx → python-docx             2. Claude (structured):         ║
║                                         "Extract from this JD:       ║
║   2. Claude (structured JSON):           required_skills[]           ║
║      "Parse this resume — return:        preferred_skills[]          ║
║       name, email, phone                 years_experience_required   ║
║       skills[]                           education_required          ║
║       years_of_experience                role_title"                 ║
║       education[]                                                    ║
║       job_titles[]"                  Returns: JobRequirements         ║
║                                                                      ║
║   Returns: CandidateProfile          3. calculate_ats_coverage():    ║
║                                         keyword frequency match      ║
║                                         (local — no AI)              ║
╚══════════════════════════════════════════════════════════════════════╝
                               │
                               ▼
╔══════════════════════════════════════════════════════════════════════╗
║  STAGE 2 — DETERMINISTIC SCORING (no AI — prevents hallucination)   ║
║                                                                      ║
║  analysis_service.compute_deterministic_scores()                     ║
║                                                                      ║
║  Skills Score (0-100):                                               ║
║    matched = required_skills ∩ candidate_skills                      ║
║    score   = matched / len(required) x 100                           ║
║                                                                      ║
║  Experience Score (0-100):                                           ║
║    score = min(candidate_years / required_years, 1.0) x 100         ║
║    Capped at 100 even if overqualified                               ║
║                                                                      ║
║  Education Score (0-100):                                            ║
║    Keyword match: bachelor/master/phd in education text              ║
║    vs job requirement level                                          ║
║                                                                      ║
║  ATS Coverage Score (0-100):                                         ║
║    JD keywords found in resume / total JD keywords x 100            ║
║                                                                      ║
║  These 4 scores are passed to Claude as anchors —                   ║
║  it builds narrative around them, not the other way around.          ║
╚══════════════════════════════════════════════════════════════════════╝
                               │
                               ▼
╔══════════════════════════════════════════════════════════════════════╗
║  STAGE 3 — AI FIT ANALYSIS + STREAMING RECOMMENDATIONS              ║
║                                                                      ║
║  Step 3a — generate_fit_analysis()                                   ║
║    Claude (JSON mode):                                               ║
║      Input: candidate profile + JD + deterministic scores            ║
║      Output: FitAnalysis JSON                                        ║
║        overall_score  → anchored to deterministic scores             ║
║        fit_level      → Excellent / Good / Fair / Poor               ║
║        strengths      → List[str]                                    ║
║        gaps           → List[SkillEvaluation]                        ║
║          each:  skill, present, importance, gap_severity             ║
║        summary        → one-paragraph assessment                     ║
║                                                                      ║
║  Step 3b — stream_recommendations()                                  ║
║    Claude (streaming):                                               ║
║      Input: FitAnalysis + candidate profile + JD                     ║
║      Streams: 5+ specific, actionable resume improvement tips        ║
║        e.g. "Add quantified AWS metrics to your DevOps role"         ║
║             "Replace 'managed' with 'reduced MTTR by 40%'"           ║
║      Yields text chunks → UI displays as they arrive                 ║
╚══════════════════════════════════════════════════════════════════════╝
                               │
                               ▼
┌──────────────────────────────────────────────────────────────────────┐
│  COMPLETE: Score bars + fit badge + streaming tips displayed         │
│  Export: Markdown or PDF report with full analysis                   │
└──────────────────────────────────────────────────────────────────────┘
```

## 📊 Scoring Dimensions

| Dimension | What It Measures | How Computed |
|---|---|---|
| **Skills Match** | % of required skills the candidate has | `matched / required × 100` |
| **Experience** | Years of experience vs job requirement | `min(candidate_yrs / required_yrs, 1.0) × 100` |
| **Education** | Education level vs job requirement | Keyword match on education text |
| **ATS Coverage** | Resume keyword density vs JD | JD terms in resume / total JD terms |

### Fit Levels

| Score | Level | Meaning |
|---|---|---|
| 75–100 | Excellent | Strong match — apply with confidence |
| 50–74 | Good | Solid match with minor gaps |
| 25–49 | Fair | Possible fit — address key gaps first |
| 0–24 | Poor | Significant mismatch — consider other roles |

## 🚀 Quick Start

### 1. Clone & install

```bash
git clone https://github.com/emran-Automation-Techlead/ai-resume-analyzer.git
cd ai-resume-analyzer
pip install -r requirements.txt
```

### 2. Configure

```bash
cp .env.example .env
```

Edit `.env`:
```env
ANTHROPIC_API_KEY=sk-ant-...
```

### 3. Launch

```bash
python app.py
```

Open **http://localhost:7862**
